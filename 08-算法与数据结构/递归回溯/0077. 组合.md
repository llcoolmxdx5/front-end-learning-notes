# [77. 组合](https://leetcode-cn.com/problems/combinations/)

## 题目

给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k` 个数的组合。

你可以按 **任何顺序** 返回答案。

示例 1：

```txt
输入：n = 4, k = 2
输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```

示例 2：

```txt
输入：n = 1, k = 1
输出：[[1]]
```

提示：

- `1 <= n <= 20`
- `1 <= k <= n`

## 难度

中等

## 标签

数组 回溯

## 解题思路

标准回溯算法

## 复杂度分析

时间复杂度：`O(C_N^K × k)`
空间复杂度：`O(n+k)=O(n)`，即递归使用栈空间的空间代价和临时数组的空间代价。
