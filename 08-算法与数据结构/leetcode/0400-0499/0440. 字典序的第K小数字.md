# [440. 字典序的第 K 小数字](https://leetcode-cn.com/problems/k-th-smallest-in-lexicographical-order/)

## 题目

给定整数 `n` 和 `k`，返回 `[1, n]` 中字典序第 `k` 小的数字。

示例 1:

```txt
输入: n = 13, k = 2
输出: 10
解释: 字典序的排列是 [1, 10, 11, 12, 13, 2, 3, 4, 5, 6, 7, 8, 9]，所以第二小的数字是 10。
```

示例 2:

```txt
输入: n = 1, k = 1
输出: 1
```

提示:

`1 <= k <= n <= 10^9`

困难
